Sound Files Needed:

1. beep.mp3 - Countdown beep (3-2-1)
2. work_done.mp3 - Work cycle complete sound
3. rest_done.mp3 - Rest cycle complete sound
4. workout_done.mp3 - Workout complete sound

Placeholder files will be created. You can replace these with custom sounds.

To generate simple beeps:
- Use Audacity (File > Generate > Tone)
- Use online tools like freesound.org
- Keep files short (< 1 second)
